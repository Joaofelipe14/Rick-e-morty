Sobre o Projeto

Este é um projeto desenvolvido com Angular 16, que aplica conceitos importantes como componentização, separação de responsabilidades, consumo de API e gestão de estado básica. Ele consome a API pública do Rick and Morty (https://rickandmortyapi.com/) para permitir a pesquisa e exibição de personagens da série.

 Funcionalidades
  • search Pesquisa de personagens por nome
  • filter_list Filtragem por status (vivo, morto, desconhecido)
  • navigate_next Navegar na lista de personagens
  • info Exibição de detalhes do personagem
  • album Exibição do total de personagens
  • favorite Extra: Adicionar e listar personagens aos favoritos
 Tecnologias Utilizadas
  • check_circle Angular 16: Framework para desenvolvimento web
  • check_circle TypeScript: Linguagem utilizada no Angular
  • check_circle Angular Material: UI Components
  • check_circle HttpClient: Para consumo da API
 API Utilizada

Os dados são consumidos da API pública do Rick and Morty: Documentação

Exemplo de requisição para listar múltiplos personagens:

GET https://rickandmortyapi.com/api/character/[1,2,3]

Estrutura do Personagem

Chave Tipo Descrição
id int O ID do personagem.
name string O nome do personagem.
status string Status do personagem ('Alive', 'Dead', 'unknown').
species string Espécie do personagem.
type string Tipo ou subspecie do personagem.
gender string Gênero do personagem ('Female', 'Male', 'Genderless', 'unknown').
origin object Nome e link para o local de origem do personagem.
location object Nome e link para o local onde o personagem foi visto por último.
image string (url) Link para a imagem do personagem (300x300px).
episode array (urls) Lista de episódios em que o personagem apareceu.
url string (url) Link para a URL do personagem.
created string Data de criação do personagem na base de dados.
